Пусть p - простое число, F_p = {0,1,...,p-1}
phi: Z/(p) -> F_p
phi([a]) = a, a from {0,...,p-1}
Определим операции +, x на F_p следующим образом:
a + b = c <=> [a] + [b] = [c]
a x b = c <=> [a] x [b] = [c]
Тогда F_p называется полем Галуа порядка p. (Также обозначается GF(p)).
Определение. Пусть k - кольцо и существует n from N, n > 0 такое что for all a from k na = 0.
Тогда наименьшее из таких чисел n называется характеристикой кольца K.
Если такого n не существует, то характеристика равна 0.

Следствие. Характеристика всякого конечного поля - простое число.

Теорема 1.47. Пусть K - коммутативное кольцо с единицей. Тогда
	(IV) если кольцо главных идеалов, то фактор-кольцо K/(p) является полем тогда и только тогда, когда p - простой элемент кольца K.
	
Пусть K - коммутативное кольцо с единицей.
1. Тогда элемент a from K называется делителем элемента b from K, если Exists c from K, такой что ac = b
2. Делители единицы называются обратимыми элементами.
3. Элементы a,b from K называются ассоциированными, если существует обратимый элемент e from K, такой что ae = b.
4. Элемент a from K называется простым, если из того, что b является делителем a следует, что b является обратимым или ассоциированным элементом.
5. Кольцо K называется кольцом главных идеалов, если оно является целостным и каждый его идеал является главным.

Многочлен.
Выражение вида sum(a_i x_i)(i from 0 to n) = a0 + a1x + a2x^2 + ... + anx^n называется многочленом над кольцом K, если a0,...,an from K, а x !from K - 
переменная.

Многочлены равны, если их ненулевые коэффициенты попарно равны.
Сложение многочленов.
a(x) = a_0 + a1x + a2x^2 + ... + a_nx^n
b(x) = b_0 + b1x + b2x^2 + ... + b_nx^n
то a(x) + b(x) = c(x), где с_i = a_i + b_i.

Умножение многочленов.
a(x) • b(x) = c(x)
c_i = sum(a_j • b_(i-j))(j from 0 to i)
Утверждение. Множество многочленов над кольцом K с операциями сложения и умножение образует кольцо. Это кольцо многочленов над K, и обозначается K[x].
Определение. Если a(x) = sum(a_i • x_i)(i from 0 to n) и a_i = 0 for all i from {0,..,n}, то a - нулевой многочлен - обозначается 0.
Пусть a(x) = sum(a_i x_i) ненулевой многочлен над K. Тогда степень a(x) deg(a) = max{i|a_i !=0 }
deg(0) = - inf.
Определение. Если deg(a) = 0, то a(x) - постоянный многочлен или константа.
Определение. Если a_n = 1, n = deg(a), то a(x) - нормированный многочлен.

Теорема 1.50
Пусть f,g from K[x]. Тогда deg(f+g) <= max{deg(f),deg(g)}
deg(f•g) <= deg(f) + deg(g)
Если K - целостное кольцо, то deg(f•g) = deg(f) + deg(g).

(2) K[x] - c единицей <=> К c единицей.
(3) K[x] - целостное <=> K - целостное.

Теорема 1.55.
Пусть f1,...,fn from F[x] - не все равные нулю над полем F.
Тогда существует единственный нормированный g from F[x], такой что
(1) g делит f_i для всех i from  {1,...,n}
(2) Если h делит все f_i, то g делит h.
И g может быть представлен в виде:
g = b1•f1 + ... + bn•fn, где b1,...,b_n from F[x].
Обозначаетя g = НОД(f1,...,f_n).
Если g = НОД(f1,f2) то иногда обозначают g = (f1,f2).

a^-1 f • g = НОК(f,g) • НОД(f,g), где a_n - старший коэффициент мн-на f•g.
НОК(f1,f2,...,fn) = НОК(f1,НОК(f2,...,fn))
Определение. Простые элементы кольца F[x] над полем F называются неприводимыми многочленами.
Простой элемент - делится только на обратимые и ассоциированные элементы.
0 = deg(f•g) = deg(f) + deg(g).
