Колебание волн
Гармонические колебания
Колебания - процессы, характеризующиеся частотой повторения во времени
Колебания называются собственными(свободными), если совершаются только за счёт первоначально сообщённой энергии.
Колебания,записываемые уравнением - s = A*cos(omega_0*t+phi) , phi - начальная фаза,A - амплитуда, omega_0*t+phi - фаза
T = 2*pi/omega_0 - период
v = 1/T - частота[Гц]
s"+(omega_0)^2*s=0 s" = d^2S/dt^2
s^~ = Ae^(i(omega_0*t+phi))
Re*S^~= S = A*cos(omega_0*t+phi)

T = m*v^2/2 = m*A^2*omega_0^2/2 * sin^2(omega_0*t+phi)
П = m*A^2*omega_0^2/2 * cos^2(omega_0*t+phi)

Гармонический осциллятор - система соврешающая одномерные колебания по закону(s"+(omega_0)^2*s=0 s" = d^2S/dt^2)
F = -kx
mx" = -k*x
x" + k/m * x = 0
omega_0 = sqrt(k/m)
T_0 = 2pi*sqrt(m/k)
T = 2*pi(l/g)

x2 = A*cos(omega_0*t+phi_2)

x = x1+x2=A*cos(omega_0*t + phi)
(A_2)^2 = (A_1)^2+(A_2)^2 + 2A_1A_2cos(phi_2 - phi_1)
tg(phi) = (A_1*sin(phi_1)+A_2*sin(phi_2))/(A_1*cos(phi_1)+A_2*cos(phi_2))
1)phi_2-phi_1 = +-2*m*pi (m=0,1,...) A = A1+A2
2)phi2-phi1 = +-(2m+1)*pi A=|A1-A2|

Сложение взаимноперпендикулярных колебаний
x^2/A^2 - 2xy/AB * cos(alpha) + y^2/b^2 = sin^2(alpha)
1)alpha = m*pi (m = 0;+-1;+-2,...) y=+-(B/A)*x
2)alpha = (2m+1)*pi/2 x^2/A^2 + y^2/B^2 = 1
