Транзисторы
Полупроводниковые приборы, предназначенные для плавного изменения электрического тока под действием слабых токов или напряжений.
Транзисторы используются в качестве основных элементов в усилителях сигналов, также в качестве электронных ключей в устройствах автоматики.И ещё
в устройствах хранения информации(кэш-память, оперативная память).
Транзисторы бывают
1.Биполярные(управляются током
2.Полевые(Управляются напряжением(эл. током))
	2.1 С управляющим p-n переходом
	2.2 С изолированным затвором
		2.2.1 Со встроенным каналом
		2.2.2 С индуцированным каналом
			2.2.2.1 С плавающим затвором(для Flash памяти)
			
Биполярный транзистор - прибор с двумя близко расположенными и взаимодействующими между собой p-n переходами. Представляет собой кристалл полупроводника
одного типа проводимости разделённый тонким слоем полупроводника с другим типом проводимости. Этот тонкий слой называется базой и играет роль управляющего электрода. Две другие области называются эмиттер и коллектор.
Конструктивные особенности:
1)Число основых носителей в эмиттере много больше чем в базе и коллекторе.
2)Ширина базы много меньше чем длина пробега основных носителей заряда.
Длина свободного пробега - расстояние, которое проходит основной носитель до рекомбинации.
Рекомбинация - процесс восстановления разорванной ковалентной связи.
1)Если U_пит !=0
U_Б = 0, то I_k = 0, так как один из p-n переходов закрыт.
2)Если U_пит !=0
U_Б != 0(прямое включение), то I_k != 0, так как один из p-n переходов закрыт.
При подаче напряжения на базу, если p-n переход будет включен в прямом направлении, то в цепи базы появится электрический ток.
* Схемы с общим эмиттером
1.Входная характеристика
	I_Б(U_Б);U_k = const
	Параметры h_1_1 = delta U_Б/delta I_Б [Ом] - входное сопротивление ~ 10^2 Ом
	h_1_2 = delta U_Б/delta U_k [Ом] - входное сопротивление ~ 10^2 Ом
2. Выходная характеристика
I_k(U_k);I_Б = const
h_2_1 = delta I_k/delta I_Б
delta I_k = h_2_1 delta I_Б
h_2_1 - коэффициент передачи по току ~ 10^2
h_2_2 = delta I_k/delta U_k [1/Om]
h_2_2 - выходная проводимость.
~10^-4 1/Ом
Замечания.
Т.к p-n переход база-эмиттер в рабочем режиме включен в прямом направлении, то его сопротивление мало. Поэтому биполярный транзистор
потребляет повышенную мощность из входной цепи. Кроме этого, открытый p-n переход обладает пониженной ёмкостью, поэтому биполярный транзистор 
обладает повышенным быстродействием.
Применение биполярных транзисторов в практических схемах информатики.
1. Управляемый делитель напряжения. Позволяет регулировать напряжение поступающее от источника тока с помощью изменения слабого входного
напряжение. В данной схеме входное напряжение - напряжение на базе.
Принципы работы:
Если R_p > R_1 => U_Б - мало => V_T - закрыт => R_кэ - велико => U_k -> U_пит
Если R_p < R_1 => U_Б - велико => V_T - откроется => R_кэ - мало => U_k -> 0
