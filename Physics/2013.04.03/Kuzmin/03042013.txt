Параметр alpha - коэффициент передачи по току = delta I_k/delta I_э (U_k = const)
r_k = delta U_k/delta I_k - дифференциальное сопротивление коллекторного перехода
r_б - сопротивление базы объёмное = (r_вх - r_э)/(1-alpha)
r_э - сопротивление эммитера = phi_T/I_э_ср
r_вх = delta U_э/delta I_э (U_k = const)

Принципы построения современных электронных устройств
1)Использование полупроводниковых электронных приборов(транзисторов,диодов и т.п)
2)Использование цифровых способов представления, обработки, передачи и хранения информации.
3)Использование универсальных цифровых вычислительных устройств(микропроцессоров и микроконтроллеров) для обработки информации
4)Поскольку внешняя информация, которую нужно обработать, представлена, как правило, в аналоговой форме, то необходимы устройства для преобразования.
Процесс аналого-цифрового преобразования.
Состоит из 3-х этапов:
1. Дискретизация по времени - разбиение непрерывно изменяющегося сигнала на ряд его мгновенных значений, взятых через одинаковые промежутки. Основной параметр - частота дискретизации(число отчётов за одну секунду).
Теорема Котельникова - Найквиста: Для однозначного восстановления непрерывного сигнала, по его мгновенным дискретным значениям, частота дискретизации должна,
как минимум, вдвое превышать максимальную частоту в его спектре. (f_0>=2Fmax)
2. Квантование по уровню - процесс замены точного мгновенного значения сигнала приближенным дискретным значением. Основной параметр - шаг квантования - 
минимальная величина на которую может изменяться сигнал. Шаг квантования зависит от максимального входного напряжения, а также от параметров процесса
кодирования. Sigma - ошибка квантования  
3. Кодирование - процесс, при котором каждому дискретному значению входного напряжения ставится в соответствие двоичное число. Основной параметр - 
разрядность - p(глубина квантования). Это число разрядов двоичного числа, которым можно закодировать ряд дискретных значений напряжения.
deltaU = U_max/2^p - 1 - шаг квантования.
Чем больше разрядность и меньше шаг квантования, следовательно, тем точнее отквантован сигнал.
#CD-audio
f_0 = 44.1 кГц
p = 16 bit
2^p-1 = 2^16 - 1 = 65535

Процесс цифро-аналогового преобразования. Состоит из двух этапов:
1. Декодирование - каждому значению цифрового кода в определённый момент времени сопоставляется определённое напряжение.
2. Фильтрация высокочастотных составляющих полученного аналогового сигнала.

Цифровые интегральные схемы.
Классификация
1. По способу кодирования
	1.1 Потенциальные - логическому нулю или логической единице соответствует определённое величина напряжения. Если U1>U0 => положительная логика. Если U1<U0
	=> отрицательная логика
	1.2 Импульсные - когда логическому нуля или единице соответствует наличие или отсутствие импульса конечной длительности.
	1.3 Имппульсно-потенциальные
2. По числу внутренних состояний
	2.1 Комбинационные - состояние выхода в каждый конкретный момент времени полностью определяется комбинацией состояний на входах
	2.2 Последовательностные - состояние выхода определяется не только состоянием входа в данный момент времени, но и предшествующим состоянием выхода
3. По способу передачи информации
	3.1 Асинхронные - воспринимают информацию сразу после её поступления на вход.
	3.2 Синхронные(тактируемые) - воспринимают входную информацию только при условии её подачи на особый вход(вход синхронизации), разрешающего сигнал.
4. По схемотехнической реализации
	4.1 ДТЛ - диодно-транзисторная логик
	4.2 ТТЛ - транзисторно-транзисторная логика - на входах многоэммитерные транзисторы, на выходе - обычные биполярные
	4.3 КМОП-логика - Использует комплементарные пары полевых транзисторов с изолированным затвором
	4.4 ЭСЛ
	4.5 И^2Л
